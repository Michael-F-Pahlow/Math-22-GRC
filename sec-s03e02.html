    Section 3.2 Episode 2: Simplifying Fractions

    Episode 2 Learning Goals.

    • Simplify fractions
      🔗
      🔗
    🔗
    🔗

    Episode 2 Key Terminology.

    • Equivalent
      🔗
      🔗
    • Simplify
      🔗
      🔗
    🔗
    🔗
    Notice that it’s possible to have two fractions that look different, but they actually represent the same amount (they are equal).
    🔗
    For example, on a tape measure, we can see that 2/8 of an inch is the same as 1/4 of an inch. These fractions are equal.
    \begin{equation*} \frac{2}{8} = \frac{1}{4} \end{equation*}
    🔗
    From a picture, we can see that 1/2 of a pizza is the same amount as 2/4 of that pizza. It’s also the same as 3/6 of that pizza, and it’s also the same as 4/8 of that pizza. All these fractions are equal.
    \begin{equation*} \frac{1}{2} = \frac{2}{4} = \frac{3}{6} = \frac{4}{8} \end{equation*}
    🔗

    Definition 3.2.3.

    Equal fractions are sometimes also called equivalent.
    🔗
    🔗
    Making Equivalent Fractions
    🔗
    There are two actions that make a fraction look different but keep it equal to what it was originally:
    1. Multiply the numerator and denominator by the same number.
      🔗
      🔗
    2. Divide the numerator and denominator by the same number (called “simplifying” because it makes the fraction simpler)
      🔗
      🔗
    For example, if I start with the fraction 1/2, and I multiply the numerator and denominator by 4:
    \begin{equation*} \frac{1\times 4}{1\times 4} = \frac{4}{8} \end{equation*}
    I get 4/8, which we saw earlier is equal to 1/2.
    🔗
    If I start with the fraction 3/6, and I divide the numerator and denominator by 3:
    \begin{equation*} \frac{3\div 3}{6\div 3} = \frac{1}{2} \end{equation*}
    I get 1/2, which we saw earlier is equal to 3/6.
    🔗

    Definition 3.2.4.

    Simplify a fraction by dividing the numerator and denominator by the same number. If no whole numbers divide both the numerator and denominator, then the fraction is said to be simplified to lowest terms.
    🔗
    🔗
    How to Fully Simplify a Fraction
    🔗
    1. Find a number that divides both the numerator and denominator of the fraction.
      🔗
      🔗
    2. Divide the numerator and denominator by this number.
      🔗
      🔗
    3. Repeat steps 1-2 until there are no more numbers that divide both numerator and denominator.
      🔗
      🔗
    🔗

    Example 3.2.5.

    Simplify the fraction
    \begin{equation*} \frac{12}{18} \end{equation*}
    🔗
    Solution 1: I notice that the number 2 divides both 12 and 18.
    \begin{equation*} \frac{12\div 2}{18\div 2} = \frac{6}{9} \end{equation*}
    I notice that the number 3 divides both 6 and 9:
    \begin{equation*} \frac{6 \div 3}{9\div 3} = \frac{2}{3} \end{equation*}
    There are no numbers that divide both 2 and 3, so the fully simplified fraction is: \(\frac{2}{3}\)
    🔗
    Solution 2: I notice that the number 3 divides both 12 and 18.
    \begin{equation*} \frac{12\div 3}{18\div 3} = \frac{4}{6} \end{equation*}
    I notice that the number 2 divides both 4 and 6:
    \begin{equation*} \frac{4\div 2}{6\div 2} = \frac{2}{3} \end{equation*}
    There are no numbers that divide both 2 and 3, so the fully simplified fraction is: \(\frac{2}{3}\)
    🔗
    🔗
