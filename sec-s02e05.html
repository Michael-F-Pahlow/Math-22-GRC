    Section 2.5 PEMDAS Integers

    Episode Learning Goals.

    • Perform multi-step calculations involving integers.
      🔗
      🔗
    🔗
    🔗
    The same rules of PEMDAS apply when working with integers
    🔗

    Example 2.5.1.

    Calculate
    \begin{equation*} -[3-4(6\div 2)^2] \end{equation*}
    🔗
    Solution: First, we work on the calculation inside the innermost parentheses, \(6\div 2\text{:}\)
    \begin{equation*} -[3-4(3)^2] \end{equation*}
    Then, we work on the calculation inside the brackets, \(3-4(3)^2\text{.}\) Applying PEMDAS here, we calculate the exponent \((3)^2\) next:
    \begin{equation*} -[3-4(9)] \end{equation*}
    We calculate the multiplication next:
    \begin{equation*} -3[3-36] \end{equation*}
    Then the subtraction:
    \begin{equation*} -[-33] \end{equation*}
    Finally, we remember that the extra - is telling us to find the opposite of -33. So our answer is
    \begin{equation*} -[-33] = 33 \end{equation*}
    🔗
    🔗

    Be Careful.

    There is a difference between
    \begin{equation*} -3^2 \quad \text{and} \quad (-3)^2 \end{equation*}
    By default, an exponent only applies to the single symbol directly underneath it. So in \(-3^2\) only the 3 is getting copied, not the \(-\text{:}\)
    \begin{equation*} -3^2 = -3\times 3 = -9 \end{equation*}
    If you want to copy the entire quantity -3, then you need to wrap -3 in parentheses before applying the exponent, like in \((-3)^2\text{:}\)
    \begin{equation*} (-3)^2 = -3 \times -3 = 9 \end{equation*}
    🔗
    🔗
