    Section 1.4 Episode 4: Order of Operations (PEMDAS)

    Episode 4 Learning Goals.

    • Perform multi-step calculations in order using PEMDAS
      🔗
      🔗
    • Recognize “hidden” parentheses / grouping, for example, when division is represented using a horizontal bar
      🔗
      🔗
    🔗
    🔗
    In problems where we have to do more than one operation (addition and multiplication, or maybe division and subtraction and exponents), we have rules for the order of those operations.
    1. Parentheses: First, do operations inside parentheses.
      🔗
      🔗
    2. Exponent: Then, calculate exponents.
      🔗
      🔗
    3. Multiplication/Division: Then, do all the multiplication and division in the problem from left to right.
      🔗
      🔗
    4. Addition/Subtraction: Then, do all the addition and subtraction in the problem from left to right.
      🔗
      🔗
    🔗
    We remember the order of operations using the mnemonic PEMDAS (“please excuse my dear Aunt Sally”).
    🔗

    Example 1.4.1.

    Calculate \(2^6 + \frac{6}{3} -2(5+3)\)
    🔗
    Solution:
    🔗
    P: First, we do the calculation in the parentheses, \(5+3=8\text{.}\)
    \begin{equation*} 2^6 + \frac{6}{3} - 2(8) \end{equation*}
    🔗
    E: Then, we calculate the exponent, \(2^6=64\text{.}\)
    \begin{equation*} 64 + \frac{6}{3} - 2(8) \end{equation*}
    🔗
    MD: Then, we do the division and multiplication, from left to right. As we read the calculation from left to right, we see the division \(\frac{6}{3}=2\text{,}\) so we calculate that next:
    \begin{equation*} 64 + 2 - 2(8) \end{equation*}
    As we continue to read the calculation from left to right, we see the multiplication \(2(8)=16\text{,}\) so we calculate that next:
    \begin{equation*} 64 + 2 - 16 \end{equation*}
    🔗
    AS: Then, we do the addition and subtraction, from left to right. As we read the calculation from left to right, we see the addition \(64+2 = 66\text{,}\) so we calculate that next:
    \begin{equation*} 66 - 16 \end{equation*}
    Finally, this subtraction is all that’s left:
    \begin{equation*} 66-16=50 \end{equation*}
    🔗
    🔗
    There are a few things that should be considered the same as parentheses when doing PEMDAS calculations.
    1. Brackets
      🔗
      Sometimes brackets [ ] are used instead of parentheses ( ) in multi-step calculations.
      \begin{equation*} 3[7-2] \end{equation*}
      So calculate the operations inside the brackets first:
      \begin{equation*} 3[7-2] = 3[5] = 15 \end{equation*}
      🔗
      🔗
    2. Division
      🔗
      When division is represented using the horizontal bar, you should pretend that there are parentheses around what’s above the bar and around what’s below the bar:
      \begin{equation*} \frac{3+7}{1+1} \hspace{0.5in} \text{should be treated like}\hspace{0.5in} \frac{(3+7)}{(1+1)} \end{equation*}
      So calculate the addition above and below the fraction bar first:
      \begin{equation*} \frac{10}{2} = 5 \end{equation*}
      🔗
      🔗
    3. Some problems contain nested parentheses / brackets. If this happens, work from the inside out, starting with the calculations in the innermost parentheses.
      🔗
      🔗
    🔗

    Example 1.4.2.

    Calculate
    \begin{equation*} 4+2[15-3(5-1)] \end{equation*}
    We start with the calculation \(5-1\) in the innermost set of parentheses:
    \begin{equation*} 4+2[15-3(4)] \end{equation*}
    Then, we work on the problem \(8-3(4)\) inside the brackets. Using PEMDAS here, we should do the multiplication \(3(4)\) before doing the subtraction.
    \begin{equation*} 4+2[15-12] \end{equation*}
    Now we complete the subtraction \(15-12\) to finish the calculation inside the brackets:
    \begin{equation*} 4+2[3] \end{equation*}
    Using PEMDAS again, we should do the multiplication before the addition:
    \begin{equation*} 4+2[3] = 4+ 6 \end{equation*}
    \begin{equation*} 4+ 6 = 10 \end{equation*}
    🔗
    🔗
