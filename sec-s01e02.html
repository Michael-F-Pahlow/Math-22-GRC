    Key Symbols.

    Multiplication
    \begin{equation*} * \hspace{1.0in} \text{as in}\,\, 5*2 \end{equation*}
    \begin{equation*} \times \hspace{1.0in} \text{as in}\,\, 5\times 2 \end{equation*}

    \begin{equation*} / \hspace{1.0in} \text{as in} \,\, 16/8 \end{equation*}
    \begin{equation*} \div \hspace{1.0in} \text{as in} \,\, 12\div 4 \end{equation*}
    🔗
    🔗

    Definition 1.2.1.

    Multiplication is the operation that corresponds to adding a number to itself repeatedly.
    🔗
    🔗
    Multiplication can be represented symbolically in many ways:
    \begin{equation*} 3\times 2 \end{equation*}
    \begin{equation*} 3\cdot 2 \end{equation*}
    \begin{equation*} 3*2 \end{equation*}
    \begin{equation*} 3(2) \end{equation*}
    \begin{equation*} (3)(2) \end{equation*}
    All of these are read aloud as “3 times 2,” and all of them mean, “add 3 copies of 2 together.”
    \begin{equation*} 3\times 2 = 2+ 2+ 2 = 6 \end{equation*}
    🔗

    Example 1.2.2.

    Calculate \(2\times 5\)
    🔗
    Solution: This is asking us to add together 2 copies of 5.
    \begin{equation*} 2\times 5 = 5 + 5 = 10 \end{equation*}
    🔗
    🔗

    Example 1.2.3.

    If you have 4 bags of apples, and each bag contains 3 apples, how many apples do you have in total?
    🔗
    Solution: Since we have several piles, each containing the same number of apples, this is a multiplication problem, specifically the problem
    \begin{equation*} 4\times 3 \end{equation*}
    So, in total we have
    \begin{equation*} 4\times 3 = 3+ 3+3+3 = 12 \,\,\text{apples} \end{equation*}
    🔗
    🔗

    Notice Something Neat.

    You can change the order of the numbers in a multiplication problem and still get the same answer!
    🔗
    \(4\times 2 = 8\) because 🟢🟢 + 🟢🟢 + 🟢🟢 + 🟢🟢 = 🟢🟢🟢🟢🟢🟢🟢🟢
    🔗
    \(2\times 4 = 8\) because 🟢🟢🟢🟢 + 🟢🟢🟢🟢 = 🟢🟢🟢🟢🟢🟢🟢🟢
    🔗
    🔗
    For problems involving larger numbers, you can use the \(\times\) button on your calculator to perform multiplication.
    🔗

    Example 1.2.4.

    You get paid $2562 every month. How much are you paid in total in 6 months?
    🔗
    Solution: Since you’re adding up multiple copies of 2562, this is a multiplication problem, specifically the problem
    \begin{equation*} 6\times 2562 \end{equation*}
    We can use the \(\times\) button on our calculator to find the answer to this problem:
    \begin{equation*} 6\times 2562 = 15,372 \end{equation*}
    So that means we are paid $15,372 in total.
    🔗
    🔗
    There are a couple of common key words that indicate that we need to perform multiplication. These include:
    • Double (multiply by specifically 2)

      🔗
    • Triple (multiply by specifically 3)
      🔗
      🔗
    • Product
      🔗
      🔗
    • Of
      🔗

    Example 1.2.5.

    What is the product of 5 and 9?
    🔗
    Solution: This is asking us to multiply 5 and 9.
    \begin{equation*} 5\times 9 = 45 \end{equation*}
    🔗
    🔗

    Example 1.2.6.

    What results when we double the number 8?
    🔗
    This is asking us to multiply 8 by 2.
    \begin{equation*} 2\times 8 = 16 \end{equation*}
    🔗
    🔗

    Definition 1.2.7.

    Division is the operation that corresponds to splitting a pile into smaller piles.
    🔗
    🔗
    Division can be represented symbolically in many ways:
    \begin{equation*} 8 \div 2 \end{equation*}
    \begin{equation*} 8/2 \end{equation*}
    \begin{equation*} \frac{8}{2} \end{equation*}
    All of these are read aloud as “8 divided by 2.”
    🔗
    There are two ways to think about 8 divided by 2:
    1. Take a pile containing 8 items and split that into 2 smaller piles. Then count the number of objects in each pile you made.
      🔗
      🟢🟢🟢🟢🟢🟢🟢🟢 → 🟢🟢🟢🟢 and 🟢🟢🟢🟢
      🔗
      There are 4 items in each pile, so \(8\div 2 = 4\)
      🔗
      🔗
    2. Take a pile containing 8 items and split that into piles so that each pile contains 2 items. Then count the number of piles you made.
      🔗
      🟢🟢🟢🟢🟢🟢🟢🟢 → 🟢🟢 and 🟢🟢 and 🟢🟢 and 🟢🟢
      🔗
      We made 4 piles, so \(8\div 2 =4\)
      🔗
      🔗
    🔗

    Example 1.2.8.

    If you have 6 cookies, and those cookies need to be shared fairly between 3 people, how many cookies can each person eat?
    🔗
    Solution: Since we are splitting a pile of 6 cookies into 3 smaller piles, this is a division problem. Specifically, this is the problem:
    \begin{equation*} \frac{6}{3} \end{equation*}
    Because 🍪🍪🍪🍪🍪🍪 → 🍪🍪 and 🍪🍪 and 🍪🍪 we see that each person gets 2 cookies. In other words,
    \begin{equation*} \frac{6}{3}=2 \end{equation*}
    🔗
    🔗

    Example 1.2.9.

    You are throwing an all-day party and you have 12 sodas. Each party guest is expected to drink 3 sodas. How many guests can you invite to the party?
    🔗
    Solution: Since we are splitting a pile of 12 sodas into portions, each containing 3 sodas, this is a division problem. Specifically, this is the problem:
    \begin{equation*} \frac{12}{3} \end{equation*}
    Because 🥤🥤🥤🥤🥤🥤🥤🥤🥤🥤🥤🥤 → 🥤🥤🥤and 🥤🥤🥤 and 🥤🥤🥤 and 🥤🥤🥤 we see that we can give sodas to 4 guests. In other words,
    \begin{equation*} \frac{12}{3}=4 \end{equation*}
    🔗
    🔗

    Notice Something Neat.

    When you divide 0 by any other number, the result is always 0.
    \begin{equation*} \frac{0}{3}=0 \end{equation*}
    Because if you have 0 items split into 3 piles, each pile will also contain 0 items!
    🔗
    🔗
    For problems involving larger numbers, you can use the / button or the \(\div\) button on your calculator to perform multiplication.
    🔗

    Example 1.2.10.

    A school is organizing a field trip. There are 168 students going on the trip. Each bus can carry 24 students. How many buses are needed to transport all the students?
    🔗
    Solution: Since splitting 168 students into piles each containing 24 students, this is a division problem, specifically the problem
    \begin{equation*} 168\div 24 \end{equation*}
    We can use the \(\div \) button on our calculator to find the answer to this problem:
    \begin{equation*} 168\div 24=7 \end{equation*}
    So that means we would need 7 buses in total.
    🔗
    🔗

    Example 1.2.11.

    What is half of 18?
    🔗
    This is asking us to divide 18 by 2:
    \begin{equation*} \frac{18}{2}=9 \end{equation*}
    🔗
    🔗

    Example 1.2.12.

    A car can drive 120 miles on 6 gallons of gas. What’s the car’s mileage in miles per gallon?
    🔗
    Solution: This is asking us to divide 120 by 6:
    \begin{equation*} \frac{120}{6}=20 \end{equation*}
